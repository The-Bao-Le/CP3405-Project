# Acceptance Criteria - Week 2

The Week 2 sprint is considered complete only when all of the following are done:

1. The team has completed three agent outputs before using any LLM:
   - Almanac Agent
   - Macro / News Agent
   - Technical Agent

2. The team has queried all four LLMs using the same prompt:
   - Claude
   - ChatGPT
   - Gemini
   - DeepSeek

3. All raw LLM responses are saved in GitHub.

4. The Multi-LLM comparison table is completed.

5. The Human Score table is completed with a clear override paragraph explaining where our team agrees or disagrees with the AI consensus.

6. The final prediction file includes:
   - SPX direction, % range, and confidence
   - NDX direction, % range, and confidence
   - IWM direction, % range, and confidence
   - 11 sector ETF calls

7. Evidence files are committed:
   - Finviz 1W screenshot
   - Yahoo Finance sector 5D screenshot
   - Annotated chart screenshot
   - Agent output files
   - LLM raw response files

8. All works are committed to GitHub before Monday.
