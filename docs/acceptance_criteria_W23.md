# Acceptance Criteria - Week 3

The Week 3 sprint is considered done when:

- The Week 3 prediction file is committed to GitHub before Sunday 7 June.
- The prediction includes SPX, NDX, and IWM direction, percentage range, and confidence level.
- Almanac, Macro, and Technical agent outputs are completed before using the LLM responses.
- Raw responses from ChatGPT, Claude, Gemini, and DeepSeek are saved in the evidence folder.
- The Multi-LLM comparison table is completed.
- The Human Score table and override paragraph are completed.
- The final prediction explains whether the team agrees or disagrees with the AI consensus.
- At least one “wild card” insight is included: something the team noticed that the AI models did not raise.
- GitHub evidence folder contains screenshots/data sources used for the prediction.
- README is updated to show Week 3 sprint status.
- PowerPoint is ready.
- Each role holder has a prepared speaking section.
